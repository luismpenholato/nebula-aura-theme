# Security Policy

## Overview

Nebula Aura Theme is a local VS Code theme extension. It applies color themes inside your editor and does not run background services.

## Data and network

- The extension does **not** collect telemetry.
- The extension does **not** connect to external services.
- The extension does **not** access files outside normal editor theme behavior.

## Reporting a vulnerability

If you believe you found a security issue, please report it via [GitHub Issues](https://github.com/luismpenholato/nebula-aura-theme/issues).

Include:

- A clear description of the issue
- Steps to reproduce, if applicable
- Your editor and extension version

## Out of scope

Color preferences, contrast suggestions and general theme feedback are not security issues. Please use GitHub Issues for normal bug reports and feature requests.
